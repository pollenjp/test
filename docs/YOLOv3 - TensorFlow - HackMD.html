TensorFlowによるYOLOv3

https://github.com/maiminh1996/YOLOv3-tensorflow

YOLOv3をTensorFlowで実装されたコードを見て読み解いていたところ,上に示した構造と異なっている箇所を発見してしまった (以下)
https://github.com/maiminh1996/YOLOv3-tensorflow

scale1とscale2を繋ぐ箇所に違いがみられる. 最初の図ではscale2からupsampleを介してscale1に矢印が伸び, scale1の Detection Result にまで達している. 一方, ２番目の図ではupsample_1にでの矢印が scale1 から scale2 に伸びている.

実装手順

  • https://github.com/maiminh1996/YOLOv3-tensorflow からクローンしてくる
git clone https://github.com/maiminh1996/YOLOv3-tensorflow.git
  • コードの中でパス指定が変数かされていない箇所があったためクローンしてきたディレクトリ名をyolo3に変更し,設定の内容も変える.
mv YOLOv3-tensorflow yolo3
vim yolo3/config.py

path = '/home/minh/PycharmProjects'

の箇所を

path = '..'

に変更

  • 公式から重みをダウンロードする. (公式のサイトにも載ってる)
wget https://pjreddie.com/media/files/yolov3.weights
  • 重みを.h5の形式に変換
    • なぜか説明がなかったが他のレポジトリの変換を利用した
    • https://github.com/qqwweee/keras-yolo3
git clone https://github.com/qqwweee/keras-yolo3.git
cd keras-yolo3
python3 convert.py yolov3.cfg yolov3.weights yolov3.h5
cp yolo.h5 ../yolo3/model/
cd ..
  • サンプル写真が欲しいので公式のgithubレポジトリをクローン
git clone https://github.com/pjreddie/darknet
  • 推論の実行
    • Entrer a check point at epoch :と聞かれたら適当な数字を入力した (例:5
    • Input image filename:と聞かれるので写真のパスを指定する(例：/../darknet/data/dog.jpg). 最初は/で始めるが絶対パスでなくてもよい.
 % python3 propagation.py image COCO output.png

コードの構造

  • Generate colors for drawing bounding boxes
    • HSV(Hue Saturation Value:色相,彩度,明度)色体系 のうち,Hueの値 ( 0∼10∼1 ) のみクラス数で等分割する(他は1.0).
    • 22.6. colorsys — 色体系間の変換 — Python 3.6.5 ドキュメント
  • placeholder作成
  • YOLOv3().feature_extractor()
    • network_function.py
    • class YOLOv3(object)
      • feature_extractor(self)
        • 以下の二つFeatures,SCALSEから成る
        • with tf.name_scope(“Features”):
        • tf.name_scope(‘SCALE’)
          • tf.name_scope(‘scale_1’):
          • tf.name_scope(‘scale_2’):
          • tf.name_scope(‘scale_3’):
      • conv2d
        • feature_extractor内部で使われている関数
        • もし独自のデータを扱いたいとき(fine-tuning)は出力部分をクラス数に応じて変化させる必要が有るためif文で場合分けしている.
        • 学習済み重みを使う際にはW()関数で抽出している
          • charger_poids.py
          • W(number_conv) (h5から抽出)
        • batch_normalizationを行うための情報の取得 B() + 𝛼α
          • B(number_conv) (h5から抽出)
